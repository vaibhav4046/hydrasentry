# HydraSentry UI Asset Brief for Claude Code

Use this folder as the complete UI asset pack.

## Visual style
Black and white only. Premium noir. No orange. No colorful gradients. Use grayscale states, glow, outlines, opacity, and motion instead of multiple colors.

## Must-use assets
- `brand/hydrasentry-mark.svg`
- `brand/hydrasentry-wordmark.svg`
- `brand/favicon.svg`
- `backgrounds/hero-noir-graph-4k.png`
- `backgrounds/noir-grid.svg`
- `backgrounds/hero-memory-tree.svg`
- `patterns/noir-noise.png`
- `tokens/tokens.css`
- `tokens/framer-motion-variants.ts`

## Button design
Primary buttons should be polished white glass on black:
- white to silver gradient
- black text
- soft white glow
- 1px white border
- hover increases brightness, not color

Secondary buttons:
- transparent black glass
- white text
- thin white border
- subtle hover fill

Do not use blue, orange, red, green, or violet for primary UI. If risk states are needed, use white intensity, labels, icons, and border patterns.

## Page mood
The app should look like a classified graph security terminal, not a generic SaaS template.
